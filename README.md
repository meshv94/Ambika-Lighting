## ğŸ¨ Customization

### Colors
The website uses a purple-pink-gold color scheme. To change colors:
- Edit gradient classes in HTML (e.g., `from-purple-600 to-pink-600`)
- Modify CSS variables in `styles.css`

### Images
Replace placeholder images:
1. Add your images to `assets/images/`
2. Update image sources in `app.js` (gallery data)
3. Replace background images in HTML

### Content
- Edit text directly in `index.html`
- Update gallery items in `app.js` (photosData and videosData arrays)
- Modify testimonials in `app.js` (testimonials array)
